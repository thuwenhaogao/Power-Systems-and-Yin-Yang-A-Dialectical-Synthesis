# Power-Systems-and-Yin-Yang-A-Dialectical-Synthesis （电力系统与阴阳的辩证统一）

有功为阳、无功为阴，二者互根互用，阴极必阳，贯穿电气工程“发—输—配—用”全过程。发电侧同步机过励发无功为阴中有阳，欠励吸无功则阴极必阳，有功由原动机定，属阳；输配侧潮流中有功沿实部流动为阳，无功沿虚部循环为阴，阴阳相依共定电压与功角；用电侧感应电机需无功建磁场为阴，有功做功为阳，阴阳相济方成能量转换。全系统以有功为体、无功为用，阴阳平衡则运行安全高效，失衡则电压崩溃、线损剧增，恰如太极调和，生生不息。
